## Description

A module to be able to create a Dice and roll it

## Table of Contents

## Installation

```bash
npm install roll-dice-quentin
```

## Usage

```ts
import Dice from "roll-dice-quentin";
const dice = new Dice();
dice.roll(6);
dice.roll(10);
dice.roll(20);
```
